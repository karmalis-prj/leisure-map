# 레저 지도 (Leisure Map)

**대한민국 레저 활동 지도 — 다이빙부터 시작합니다.**

전국 다이빙샵 125곳을 실측 조사로 수집·검증한 무료 공개 데이터입니다.

🔗 **https://karmalis-prj.github.io/leisure-map/**

## 담긴 것

- 전국 21개 지역 다이빙샵: 가격·전화·영업시간·리뷰 특성(강습우수/시설우수/초보친화 등)
- 지역별 수중환경(시야·조류·수온·수심)과 계절별 다이빙 적합도
- 샵 주변 맛집·숙박·볼거리(관광명소) 통합
- 1박 2일 여행 코스 자동 추천

## 데이터 원칙

- **실측 우선**: 모든 샵은 출처(source)와 검증 상태(verified)를 명시합니다.
- **신선도 관리**: 매월 자동 재검증으로 폐업·전화 변경을 감지하고 `lastVerified` 스탬프를 갱신합니다.
- **프라이버시**: 리뷰 원문은 저장하지 않습니다. 네이버가 공개 집계한 키워드 통계만 사용합니다.
- **미확인은 미확인으로**: 확인 못 한 정보는 추정치로 채우지 않고 "미확인"으로 표시합니다.

## 로드맵

다이빙 → 프리다이빙·서핑·패들보드 등 수상 레저 → 전국 레저 지도로 확장 예정.
